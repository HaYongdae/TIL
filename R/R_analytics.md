# R_통계 분석

> 검정 - 데이터 간의 관계를 검토하는 과정

```bash
단일 집단 - 전. 후 표본 - 비율의 차이 검정 - binom.test() (유의순준과 유의확률)
단일 집단 - 평균의 정규분포 검정 -shapiro.test() (평균의 비교를 위해)

-> 확인 후 (평균) 비교 검정
단일 집단 - 평균이 정규분포를 따르는 경우 t.test() (유의확률과 t검정통계량)
단일 집단 - 평균이 정규분포를 따르지 않는 경우 wilcox.text()

-> (동질성) 비교 검정
두 집단 - 비율 검정 - prop.test() (독립표본 이항분포 비율 검정)
두 집단 - 평균 검정 -> 두 집단의 평균의 정규분포가 동일한지 검정 (동질성 검정 -var.test() )
두 집단 - 평균이 정규분포를 따르는 경우 t.test()
두 집단 - 평균이 정규분포를 따르지 않는 경우 wilcox.text()

대응 두 집단 = ( EX_교수법 전후 동일 대상의 서로 다른 점수)의 평균 차이 비교
대응 두 집단 - 평균의 정규분포가 동일한지 검정 (동질성 검정 -var.test() )
대응 두 집단 - 평균이 정규분포를 따르는 경우 t.test()
대응 두 집단 - 평균이 정규분포를 따르지 않는 경우 wilcox.text()

세 집단 - 비율 검정 - prop.test()
세 집단 - 평균 검정 - 분산분석, F검정
세 집단 - 평균의 정규분포가 동일한지 검정 (동질성 검정 - bartlett.test())
세 집단 - 평균이 정규분포를 따르는 경우 aov()
세 집단 - 정규분포를 따르지 않는 경우 kruskal.test()

사후 검정 - TukeyHSD()

요인 분석 - 변수간의 관계를 분석 - 상관/회귀분석의 설명변수(독립변수)로 활용하기 위한 근거

변수의 주요 성분 분석 요인수 -
1. 주성분 분석 - prcomp()
2. 고유값(변수의 상관계수 행렬)으로 요인수 분석 - eigen()

변수들간의 상관관계 분석으로 요인 분석 - 변수들간의 상관성을 이용해서 공통요인 추출 
factanal(dataset, factors="" , rotation="", scores="")

다중회귀분석 (y는 1개, x는 여러개)
1. 셈플 - train, test (7:3 분리)
2. vif 로 다중공선성 확인
3. lm(formula = y ~ x..., data)			- 모델 생성
4. predict(lm모형, train)					- 모델의 예측치 출력
5. cor(predict값, test)					- 모델 정확성 검사

로지스틱 회귀분석 ( y가 2개, x가 여러개)
1. 로짓 변환
2. glm(formula = y ~ x..., data)
3. predict에 response를 적용 ( 0.5 > 와 0.5 < 를 0,1로 구분)
4. 테이블로 나눈다.

분류 분석 (y,x가 모두 여러개)
- 의사결정나무
- 렌덤포레스트
```
