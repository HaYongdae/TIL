# MarkDown 문법

마크다운 문법을 활용하여 git을 쉽게 관리 할 수 있다.

주로 typora 에디터를 다운 받아 활용한다 ([다운로드](https://typora.io/))

## 1. 제목

각종 마크다운 문법

- "#" 활용 - 제목(개략에 표기되는) 설정

- tap 과 shift tap 으로 앞여백 조정



## 2. 폰트 변경

- 기울임  : *로 양쪽 감싸기  => *안녕*
- 볼드체 : * 두개로 양쪽 감싸기 => **안녕**
- 기울임 + 볼드체 : * 세개로 양쪽 감싸기 => ***안녕***
