# **1. 제목 작성**

"#"을 쓰면 글씨 크기가 커진다. (하나씩 더하면 더할수록 글씨 크기가 점점 작아진다.)

●     #은 대제목

●     ##은 중제목

●     ###은 소제목

# **3. 강조 표현**

*코드블락을 쓸때는* **```***을 활용한다*

\1.    강조 : "**"로 텍스트를 감싸면 (볼드체)

\2.    기울임 : "*"로 텍스트를 감싸면 (기울임체)

\3.    강조 + 기울임 : "***"로 텍스트를 감싸면 (볼드+기울임체)
